# Bookshelf API

REST-сервис для каталога книг с многомерной системой рейтинга, построенный на FastAPI.

---

## О проекте

**Bookshelf API** позволяет пользователям вести каталог книг, оставлять подробные отзывы с оценками и получать объективный рейтинг произведений.

Проект выполнен в рамках итогового задания по FastAPI и соответствует всем требованиям:

- Реализация CRUD-операций с реляционной БД
- Аутентификация и авторизация пользователей (JWT)
- Решение бизнес-задачи с алгоритмической логикой
- Автоматическое тестирование (pytest + TestClient)
- Чистый код (pylint 10.00/10)

---

## Бизнес-задача

Каждый пользователь может оценить книгу по **четырём критериям** (от 1 до 5):

| Критерий | Описание |
|---|---|
| `plot` | Качество сюжета |
| `style` | Стиль письма автора |
| `atmosphere` | Атмосфера произведения |
| `characters` | Проработка персонажей |

Сервис автоматически рассчитывает средние значения по каждому параметру и **общий рейтинг** книги по формуле:

```
overall_rating = (avg_plot + avg_style + avg_atmosphere + avg_characters) / 4
```

Эндпоинт `/ratings/top` возвращает топ книг, отсортированный по этому рейтингу.

## Структура базы данных

Используется SQLite (файл `bookshelf.db` создаётся автоматически при первом запуске).

| Таблица | Описание |
|---|---|
| `users` | Пользователи |
| `books` | Книги |
| `reviews` | Отзывы |

```
users 1───N reviews N───1 books
```

## Запуск проекта

```bash
git clone https://github.com/MariaLavrinenko/bookshelf_api_rep
cd bookshelf

python -m venv .venv
.venv\Scripts\activate          # Windows

pip install -r requirements.txt

uvicorn app.main:app --reload
```

## Эндпоинты

| Метод | Эндпоинт | Авторизация | Описание |
|---|---|-------------|---|
| `POST` | `/auth/register` | —           | Регистрация нового пользователя |
| `POST` | `/auth/login` | —           | Вход в систему (получение JWT-токена) |
| `GET` | `/books/` | —           | Получить список всех книг |
| `POST` | `/books/` | +           | Добавить новую книгу |
| `GET` | `/books/{book_id}` | —           | Получить одну книгу по ID |
| `PUT` | `/books/{book_id}` | +           | Обновить данные книги |
| `DELETE` | `/books/{book_id}` | +           | Удалить книгу |
| `GET` | `/books/{book_id}/reviews/` | —           | Получить все отзывы к книге |
| `POST` | `/books/{book_id}/reviews/` | +           | Оставить отзыв |
| `DELETE` | `/books/{book_id}/reviews/{review_id}` | +           | Удалить свой отзыв |
| `GET` | `/ratings/books/{book_id}` | —           | Детальный рейтинг одной книги |
| `GET` | `/ratings/top` | —           | Топ книг по общему рейтингу |
